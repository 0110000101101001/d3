## <tt>select(<i>s</i>)</tt>

![select](select.png)

Selects a descendent of each node in the current selection that matches the
selector <i>s</i>.

## <tt>d3.select(<i>s</i>)</tt>

![select](select-init.png)

Selects a descendent of the current document that matches the selector <i>s</i>.
